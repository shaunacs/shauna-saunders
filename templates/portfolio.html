Friendship Matchmaking Platform

A full-stack web application built for a real friendship matchmaking business, actively used in production.

Key Features
  • Member registration with email verification and account management
  • Multi-section compatibility assessment covering personality, communication style, values, and social preferences — with personalized insights generated from responses
  • Admin-curated matching workflow with individualized introduction emails written for each person in a match
  • Member dashboard showing current matches, assessment results, and upcoming events
  • Full event management system with RSVP tracking, capacity limits, and flexible pricing
  • Tablet-friendly event day check-in kiosk for use by volunteers without requiring an admin login
  • Interactive in-event speed-friending facilitation tool for admins to coordinate live pairings
  • Community partner portal for submitting assessments on behalf of prospective members
  • Admin content management for the public-facing services page
  • Automated emails at every key member touchpoint
